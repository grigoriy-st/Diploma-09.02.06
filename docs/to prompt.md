# TO PROMPT

Пишу преддипломный проект по сетевому и системному администрирование 09.02.06.
Название проекта "Проектирование КС в военкомате"

## Требования заказчика

- ЛВС без доступа в интернет
- Сервер файлообменник FreeNAS, возможность одновременного редактирования документов
- Общий доступ к сетевому принтеру
- Должна быть связность второго и пятого этажа (т.е. с пятого этажа был доступ к FreeNAS-хранилищу).
- Отечественное оборудование Eltex

## Информация об объекте

- Часть сети (часть второго этажа) проложена компанием Ростелеком через кабель-канал и имеет доступ в Интеренет, трогать эту сеть нельзя.
- Имеются кабельные туннели
- ОС различные: Astra linux, Windows XP, Windows 10
- 32 пк на втором этаже и 10 на пятом
- Кабинеты:
  - Кабинет полковника
  - Техническое помещение
  - Инвентарная команата
  - Отдел социального обеспечения
  - Кабинет призывной комиссии
  - Отдел контроля:
    - административно-хозяйственное отделение (АХО);
    - финансовый отдел;
    - гражданский отдел

## Для реализации

- Создать беспроводную ЛВС WDS with AP без доступа в интернет
  - 3 точки доступа на втором этаже и одна на пятом.
  - на втором этаже точки доступа подключены по WDS with AP, мастер точка подключена к управляемому комутатору
  - Точка с 5го этажа подключена по кабелю к коммутатору на втором этаже
- Отказоустойчивость LACP и PortSecurity
- VLAN
  - Группа «Polkovnik» — начальник военкомата (кабинет полковника). Члены группы имеют полный доступ на чтение всех папок и право на запись в папку «Приказы».
  - Группа «Social» — четыре сотрудника отдела социального обеспечения. Полный доступ к папке «СоцОбеспечение», доступ на чтение к общедоступным документам.
  - Группа «Finance» — три сотрудника финансового отдела. Полный доступ к папке «Финансы», доступ на чтение к папке «Отчёты» (общая папка для руководства).
  - Группа «AllStaff» — все пользователи (автоматическая группа). Доступ на чтение к общедоступной папке «Public» и право печати на сетевом принтере.
- Сервер с:
  - TrueNAS SCALE c RAID1 c ZFS
  - Zabbix с SNMP
  - Samba
